# 💊 PharmaSys - Sistema de Gestión

Sistema desarrollado en Python usando arquitectura MVC para la gestión de:

- Medicamentos
- Proveedores
- Clientes
- Principios Activos

---

## 🚀 Tecnologías usadas

- Python 3
- Tkinter (Interfaz gráfica)
- MySQL (Base de datos)
- Pillow (manejo de imágenes)
- tkcalendar (fechas)
- openpyxl (exportar Excel)
- reportlab (exportar PDF)

---

## 📂 Estructura del proyecto

- controllers/ → lógica de negocio
- models/ → acceso a base de datos
- views/ → interfaz gráfica
- utils/ → validaciones, exportaciones
- assets/ → imágenes e íconos

---

## ⚙️ Funcionalidades

✔ CRUD completo en todos los módulos  
✔ Validación de datos (email, números)  
✔ Exportación a Excel y PDF  
✔ Manejo de imágenes  
✔ Selector de fechas  
✔ Interfaz gráfica con Tkinter  
✔ Arquitectura MVC  

---

## ▶️ Cómo ejecutar

1. Instalar dependencias:
pip install -r requirements.txt
2. Ejecutar:
python main.py
